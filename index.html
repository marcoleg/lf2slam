LF²SLAM: Learning-based Features For Visual SLAM

Abstract

LF²SLAM is a hybrid visual odometry approach that integrates robust, data-driven feature extraction into a standard geometric visual SLAM pipeline. By combining learned features with classical optimization, LF²SLAM achieves improved generalization capabilities and robustness in challenging real-world and simulated scenarios.
